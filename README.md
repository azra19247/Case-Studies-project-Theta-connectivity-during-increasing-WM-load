# Case-Studies-project-Theta-connectivity-during-increasing-WM-load
This repository contains a data processing script and an analysis script used to analyse EEG data recorded during an auditory a digit-span WM task. It also contains the final report of the project.

The processing script filters the data files, runs ICA to remove eye-blink components and computes the debiased wPLI as a measure of fronto-parietal theta-band connectivity. 

The analysis script tests whether fronto-parietal wPLI shows an inverted U-shape acros digit positions within a sequence.
